# Modelo

Template docker compose config

## Uso

Para usar essa configuração siga os passos abaixo

- Crie seu arquivo `.env` com suas configurações com base no `.env.example`

```bash
  cp .env.example .env
```

- Execute o docker para subir os containers
```bash
  docker compose up -d
```
